## 1-2 分钟演示视频脚本

### 总时长: 约 90 秒

---

**开场 (0:00 - 0:10)**

画面：打开 Chrome，书签栏乱糟糟的

旁白：你的 Chrome 书签是不是也这样？越来越多，越来越乱，找个东西要翻半天？

**导航模式演示 (0:10 - 0:30)**

画面：Pinmark 的 Grid View 默认界面

旁白：打开之后，默认是导航模式。你的所有书签文件夹会以多列卡片的形式展示，每个书签都能看到标题和 favicon，一目了然。

旁白：点击左侧的文件夹可以切换，子文件夹会展示在旁边的列里，操作很流畅。

**列表模式演示 (0:30 - 0:45)**

画面：切换到列表模式（点击 Header 的切换按钮）

旁白：如果你习惯传统的方式，可以一键切换到列表模式。左侧是文件夹树，右侧是书签列表，简洁清晰。

画面：选中书签，展示复选框和工具栏

旁白：支持多选、批量删除、批量移动到其他文件夹。工具栏还会显示空文件夹和重复书签的数量。

---

**搜索演示 (0:45 - 0:55)**

画面：⌘F 聚焦搜索框，输入关键词

旁白：按 ⌘F 快速聚焦搜索，实时按标题和 URL 过滤，几百个书签也能秒级定位。

旁白：误删了也不要紧，这里有撤销按钮，一键恢复。

### 拍摄提示

1. **清晰度**：录制时分辨率至少 1080p
2. **局部放大**：操作细节可以适当放大，让观众看清点击位置
3. **节奏**：每个功能展示 5-8 秒，不要在一个画面上停留太久
4. **背景**：使用干净的桌面背景，建议开启 macOS 的"勿扰模式"避免通知弹出
5. **鼠标高亮**：建议开启鼠标点击高亮效果（macOS 可用 Mousepose 或系统自带的"显示鼠标指针大小"）
6. **画中画**：如果有条件，可以在角落加一个画中画摄像头画面，增加亲和力
7. **B站风格**：开头前 3 秒要足够吸引人，B站用户跳过早，建议第一句话直接抛出痛点
