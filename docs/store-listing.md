## 3. Detailed Description (English)

**Title:** Pinmark — Smart Bookmark Manager for Chrome

Pinmark is a minimal yet powerful bookmark manager that helps you organize, browse, search, and clean up your Chrome bookmarks with an intuitive interface.

**Two View Modes:**
- **Grid View** — Multi-column card layout inspired by browser speed dials. Browse folders as columns, with bookmarks displayed as cards. Perfect for quick visual scanning.
- **List View** — Classic sidebar folder tree + bookmark list. Navigate your bookmark hierarchy the traditional way, with a clean reading list layout.

**Key Features:**
- **Drag & Drop** — Reorganize bookmarks and folders with drag and drop. Supports batch drag from multi-select.
- **Full-text Search** — Real-time search by title and URL. Press ⌘F to focus the search bar instantly.
- **Batch Operations** — Multi-select with checkboxes, batch delete, batch move to any folder with a folder picker dialog.
- **Undo Delete** — Accidental deletion? A toast notification appears with an undo button to restore bookmarks or folders.
- **Dark Mode** — Toggle dark/light theme with one click. Your preference is automatically saved.
- **Open All** — Right-click any folder and choose "Open All" to open every bookmark inside in new tabs.
- **Sort by Time** — Group bookmarks by creation time (today, yesterday, this week, last month, etc.) in grid view.
- **Duplicate Detection** — Automatically identifies bookmarks with the same URL so you can clean up duplicates.
- **Empty Folder Detection** — See which folders are empty and clean them up.
- **Smart Folder Management** — Create new folders and subfolders via right-click context menu or dedicated button.
- **Keyboard Shortcuts** — ⌘F search, ⌘A select all, Delete remove, Esc close menus.

**Why Pinmark?**
Chrome's built-in bookmark manager is basic. If you have hundreds or thousands of bookmarks, Pinmark gives you the tools to take control — visual browsing, powerful search, and efficient cleanup, all in one clean interface.

## 4. 详细描述 (中文)

**标题：** Pinmark — 智能 Chrome 书签管理器

Pinmark 是一款极简而强大的 Chrome 书签管理器，通过直观的界面帮助你导航、搜索、整理和清理书签。

**两种视图模式：**
- **导航模式** — 多列卡片式布局，类似浏览器首页。文件夹以列形式展示，书签显示为卡片，适合快速视觉浏览。
- **列表模式** — 经典侧边栏文件夹树 + 书签列表，以传统方式浏览书签层级结构。

**主要功能：**
- **拖拽整理** — 书签和文件夹均可拖拽移动，支持多选批量拖拽
- **全文搜索** — 按标题和 URL 实时搜索，⌘F 快捷键快速聚焦
- **批量操作** — 复选框多选、批量删除、批量移动到任意文件夹
- **撤销删除** — 误删除后可通过 Toast 通知一键撤销恢复
- **深色模式** — 一键切换深色/浅色主题，自动记忆偏好设置
- **一键打开全部** — 右键文件夹 → 打开全部，批量打开所有书签
- **按时间排序** — 按收藏时间分组查看（今天、昨天、本周、上月等）
- **查重检测** — 自动识别相同 URL 的重复书签
- **空文件夹检测** — 发现并清理空文件夹
- **智能文件夹管理** — 通过右键菜单或按钮新建文件夹和子文件夹
- **键盘快捷键** — ⌘F 搜索 · ⌘A 全选 · Delete 删除 · Esc 关闭菜单

**为什么选择 Pinmark？**
Chrome 自带的书签管理器功能有限。如果你有数百甚至上千个书签，Pinmark 为你提供强大的管理工具 — 可视化浏览、高效搜索、快速清理，一切在一个简洁的界面中完成。
